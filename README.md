## Getting Started

First, install dependencies
```bash
npm install
```

Next, add your API keys from TMDB and setup your .env file
```bash
mv .env.local.example .env.local
```

```dotenv
API_KEY='ADD_ME'
API_TOKEN='ADD_ME'
```

Next, run the development server:

```bash
npm run dev
```
